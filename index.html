随钻高速信息传输关键技术

High-Speed Downhole Information Transmission

中石化"揭榜挂帅"项目
SINOPEC "Jiebai Guashuai" Project

MATLAB 多源泵噪声模型 + 卡尔曼滤波与 LMS/RLS 自适应滤波，噪声重构精度 >99%，信噪比提升 15dB+。主导 App Designer 实时处理软件——1s 延迟，效率提升 90%+。"界面美观、操作直观"评价。

MATLAB multi-source pump noise model + Kalman/LMS/RLS adaptive filtering: >99% noise reconstruction accuracy, 15dB+ SNR improvement. Led App Designer real-time software — 1s display latency, 90%+ efficiency gain.

2022–2024 · Algorithm Dev

高速水声通信技术

Underwater Acoustic Communication

国家杰出青年科学基金项目 / National Science Fund

STFT + 陷波滤波 + MVDR 波束成形抑制推进器噪声，SNR 提升 10dB+，误码率低于 1e-5。Verilog/FPGA 设计 SNR 自适应帧同步，舟山海试零丢帧。

STFT + notch filtering + MVDR beamforming suppressed thruster noise: 10dB+ SNR, BER below 1e-5. Verilog/FPGA SNR-adaptive frame sync, zero frame-miss in 5km sea trial.
